# Job Portal

A full-stack Job Portal application built with React, Tailwind CSS, Supabase, Clerk Authentication, and Shadcn UI.

## Features

* User Authentication
* Job Listings
* Save Jobs
* Post Jobs
* Recruiter Dashboard
* Responsive Design

## Tech Stack

* React.js
* Tailwind CSS
* Supabase
* Clerk
* Shadcn UI
* React Router

## Installation

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file and add the required environment variables.
